# Production-Scale Voice Assistant

Think Alexa + Siri + ChatGPT Voice.

## Goal

Build a production-scale voice assistant with:

- Microphone audio capture
- Voice Activity Detection
- Noise suppression
- Streaming ASR
- Intent classification
- Entity recognition
- Dialogue manager
- LLM agent
- Tool calling
- Text-to-speech
- Audio streaming
- Scalable backend with Kafka, Redis, Triton, Kubernetes
